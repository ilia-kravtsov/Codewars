<h2><a href=https://www.codewars.com/kata/58b8d22560873d9068000085/train/javascript target="_blank">Simple Fun #178: Faulty Odometer</a></h2><h3>5 kyu</h3><h2 id="task">Task</h2><p> You are given a car odometer which displays the miles traveled as an integer. </p><p> The odometer has a defect, however: it proceeds from digit <code>3</code> to digit <code>5</code> always skipping the digit <code>4</code>. This defect shows up in all positions (ones, tens, hundreds, etc). </p><p> For example, if the odometer displays <code>15339</code> and the car travels another mile, the odometer changes to <code>15350</code> (instead of <code>15340</code>).</p><p> Your task is to calculate the real distance, according The number the odometer shows.</p><h2 id="example">Example</h2><p> For <code>n = 13</code> the output should be <code>12</code>(4 skipped).</p><p> For <code>n = 15</code> the output should be <code>13</code>(4 and 14 skipped).</p><p> For <code>n = 2003</code> the output should be <code>1461</code>.</p><h2 id="inputoutput">Input/Output</h2><ul><li><code>[input]</code> integer <code>n</code></li></ul><p>  The number the odometer shows.</p><p>  <code>1 &lt;= n &lt;= 999999999</code></p><ul><li><code>[output]</code> an integer</li></ul><p>  The real distance.</p>
