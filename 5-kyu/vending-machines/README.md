<h2><a href=https://www.codewars.com/kata/526b1dd476952c04cf0000ab/train/javascript target="_blank">Vending Machines</a></h2><h3>5 kyu</h3><blockquote><p>You are in a room with 100 vending machines, each one numbered 0 to 99.  Each vending machine has 100 candy bars inside.  For 99 of the vending machines, all of their candy bars weigh 100 grams, but one special machine has candy bars that weigh 101 grams.</p><p>Candy bars you vend are put into a pile, but you can only weigh the pile at most 10 times.</p><p>Determine the number of the machine with the heavy candy bars.</p></blockquote><p>In this kata, you must solve this puzzle by coding the solution in the function <code>findSpecialIdx</code>.</p><p>The function takes a <code>VendingMachines</code> object; call it <code>vms</code>.</p><ul><li>To make the <em>i</em>th machine vend, run <code>vms[i].vend()</code>; <em>i</em> can range from 0 to 99</li><li>To weigh the pile, run <code>vms.weigh()</code></li></ul><p>The index of the special machine is random, and different for each <code>VendingMachines</code> object.  The function must return the index of the machine with the heavy candy bars.</p>
