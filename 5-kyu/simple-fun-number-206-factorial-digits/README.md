<h2><a href=https://www.codewars.com/kata/58fea5baf3dff03a6e000102/train/javascript target="_blank">Simple Fun #206: Factorial Digits</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p>Given an integer <code>n</code>, lying in range <code>[0; 1_000_000]</code>, calculate the number of digits in <code>n!</code> (factorial of <code>n</code>).</p><h3 id="example">Example</h3><ul><li>For <code>n = 0</code>, the output should be <code>1</code> because <code>0! = 1</code> has 1 digit;</li><li>For <code>n = 4</code>, the output should be <code>2</code> because <code>4! = 24</code> has 2 digits;</li><li>For <code>n = 10</code>, the output should be <code>7</code> because <code>10! = 3628800</code> has 7 digits.</li></ul>
