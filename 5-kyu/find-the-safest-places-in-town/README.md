<h2><a href=https://www.codewars.com/kata/5dd82b7cd3d6c100109cb4ed/train/javascript target="_blank">Find the safest places in town</a></h2><h3>5 kyu</h3><p><a href="https://preview.mailerlite.com/d5g0h3/1389145903520025821/l1w0/" data-turbolinks="false" target="_blank"><img title="source: imgur.com" src="https://i.imgur.com/ta6gv1i.png?1"></a></p><p><a href="https://en.wikipedia.org/wiki/Laura_Bassi" data-turbolinks="false" target="_blank">Laura Bassi</a> was the first female professor at a European university.<br>Despite her immense intellect, she was not always allowed to lecture publicly.</p><p>One day a professor with very strong beliefs against women in academia sent some <code>agents</code> to find Bassi and end her career.</p><p>Help her escape by telling her the safest places in town!</p><h2>Task</h2><p>Implement the function <code>advice(agents, n)</code> where</p><ul><li><code>agents</code> is an array of agent coordinates.</li><li><code>n</code> defines the size of the city that Bassi needs to hide in, in other words the side length of the square grid.</li></ul><p>The function should return a list of coordinates that are the furthest away (by Manhattan distance) from all agents.</p><p>As an example, say you have a 6x6 map, and agents at locations</p><pre style="display: none;"><code class="language-python">[(<span class="cm-number">0</span>, <span class="cm-number">0</span>), (<span class="cm-number">1</span>, <span class="cm-number">5</span>), (<span class="cm-number">5</span>, <span class="cm-number">1</span>)]</code></pre><pre><code class="language-javascript">[[<span class="cm-number">0</span>, <span class="cm-number">0</span>], [<span class="cm-number">1</span>, <span class="cm-number">5</span>], [<span class="cm-number">5</span>, <span class="cm-number">1</span>]]</code></pre><pre style="display: none;"><code class="language-haskell">[(<span class="cm-number">0</span>, <span class="cm-number">0</span>), (<span class="cm-number">1</span>, <span class="cm-number">5</span>), (<span class="cm-number">5</span>, <span class="cm-number">1</span>)]</code></pre><p>The distances to the nearest agent look like this.</p><p><img src="https://github.com/falcowinkler/falcowinkler.github.io/raw/master/resources/codewars/safest.png"></p><p>The safest spaces are the ones with distance <code>4</code>, marked in bright red. So the function should return</p><pre style="display: none;"><code class="language-python">[(<span class="cm-number">2</span>, <span class="cm-number">2</span>), (<span class="cm-number">3</span>, <span class="cm-number">3</span>), (<span class="cm-number">4</span>, <span class="cm-number">4</span>), (<span class="cm-number">5</span>, <span class="cm-number">5</span>)]</code></pre><pre><code class="language-javascript">[[<span class="cm-number">2</span>, <span class="cm-number">2</span>], [<span class="cm-number">3</span>, <span class="cm-number">3</span>], [<span class="cm-number">4</span>, <span class="cm-number">4</span>], [<span class="cm-number">5</span>, <span class="cm-number">5</span>]]</code></pre><pre style="display: none;"><code class="language-haskell">[(<span class="cm-number">2</span>, <span class="cm-number">2</span>), (<span class="cm-number">3</span>, <span class="cm-number">3</span>), (<span class="cm-number">4</span>, <span class="cm-number">4</span>), (<span class="cm-number">5</span>, <span class="cm-number">5</span>)]</code></pre><p>in any order.</p><p>Edge cases:</p><ul><li>If there is an agent on every grid cell, there is no safe space, so return an empty list.</li><li>If there are no agents, then every cell is a safe spaces, so return all coordinates.</li><li>if <code>n</code> is <code>0</code>, return an empty list.</li><li>If agent coordinates are outside of the map, they are simply not considered.</li><li>There are no duplicate agents on the same square.</li></ul><h2>Performance</h2><p>You will probably not pass the tests if you use a brute-force solution.In other words, you need to do better than <code>O(agents*n*n)</code>.</p><p>This kata is inspired by ThoughtWorks' <a href="https://github.com/Fun-Coding-Challenges/ada-lovelace-coding-challenge" data-turbolinks="false" target="_blank">coding challenge</a></p>
