<h2><a href=https://www.codewars.com/kata/52af840e0b1d45d45f000078/train/javascript target="_blank">Definitely Naughty or Probably Nice?</a></h2><h3>5 kyu</h3><p>Santa Claus has always kept a list of who's been bad and who's been good.  There are 7 billion people on the planet now.  The lists are getting unwieldy.  Santa has decided to use a Bloom Filter this year instead.</p><p>With a Bloom Filter, you can use a much less memory if you are willing to trade-off accuracy.  The Bloom Filter has a method <code>add(str)</code> and a method <code>check(str)</code>.  Throughout the year, Santa will <code>add()</code> the names of those who have been nice.  On Christmas Eve, he will <code>check()</code> each child's name.  If a name has been added, then <code>check()</code> will definitely return true.  If a name has not been added, <code>check()</code> will probably (but not always) return <code>false</code>.</p><p>A Bloom Filter contains a boolean array and uses multiple hash functions.  The boolean array is of a given size and initialized to all false.</p><p>During <code>add(str)</code>, each hash function is invoked against the passed in <code>str</code>, each returning a number <code>n</code>. The array element at <code>n % array.length</code> is set to true for each <code>n</code>.</p><p>During <code>check(str)</code>, each hash function is invoked again, each returning an <code>n</code>. If all of the array elements at the  <code>n % array.length</code> positions are set to true, then we assume the value was added to the list and return true. If any element is false, we return false.</p><p>The Bloom Filter will be constructed as:</p><pre><code class="language-javascript"><span class="cm-keyword">new</span> <span class="cm-variable">BloomFilter</span>( <span class="cm-variable">arraySize</span>, <span class="cm-variable">hashFn1</span>, <span class="cm-variable">hashFn2</span>, <span class="cm-variable">hashFn3</span>, <span class="cm-meta">...</span> );</code></pre><pre style="display: none;"><code class="language-coffeescript"><span class="cm-keyword">new</span> <span class="cm-variable">BloomFilter</span> <span class="cm-variable">arraySize</span><span class="cm-punctuation">,</span> <span class="cm-variable">hashFn1</span><span class="cm-punctuation">,</span> <span class="cm-variable">hashFn2</span><span class="cm-punctuation">,</span> <span class="cm-variable">hashFn3</span><span class="cm-punctuation">,</span> <span class="cm-punctuation">...</span></code></pre>
