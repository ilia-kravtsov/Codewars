<h2><a href=https://www.codewars.com/kata/57feb00f08d102352400026e/train/javascript target="_blank">Airport Arrivals/Departures - #1</a></h2><h3>5 kyu</h3><p>You are at the airport staring blankly at the arrivals/departures flap display...</p><div style="width:75%"><img src="http://www.airport-arrivals-departures.com/img/meta/1200_630_arrivals-departures.png"></div><h1 id="how-it-works">How it works</h1><p>You notice that each flap character is on some kind of a rotor and the order of characters on each rotor is:</p><p><code>ABCDEFGHIJKLMNOPQRSTUVWXYZ ?!@#&amp;()|&lt;&gt;.:=-+*/0123456789</code></p><p>And after a long while you deduce that the display works like this:</p><ul><li><p>Starting from the left, all rotors (from the current one to the end of the line) flap together until the left-most rotor character is correct.</p></li><li><p>Then the mechanism advances by 1 rotor to the right...</p></li><li><p>...REPEAT this rotor procedure until the whole line is updated</p></li><li><p>...REPEAT this line procedure from top to bottom until the whole display is updated</p></li></ul><h1 id="example">Example</h1><p>Consider a flap display with 3 rotors and 1 line which currently spells <code>CAT</code></p><hr><span style="color:red">Step 1</span>&nbsp;&nbsp;(current rotor is 1)<ul><li>Flap x 1</li><li>Now line says <code>DBU</code></li></ul><hr><span style="color:red">Step 2</span>&nbsp;&nbsp;(current rotor is 2)<ul><li>Flap x 13</li><li>Now line says <code>DO)</code></li></ul><hr><span style="color:red">Step 3</span>&nbsp;&nbsp;(current rotor is 3)<ul><li>Flap x 27</li><li>Now line says <code>DOG</code></li></ul><hr><p><em>This can be represented as</em></p><pre><code>lines  // array of strings. Each string is a display line of the initial configurationrotors // array of array-of-rotor-values. Each array-of-rotor-values is applied to the corresponding display lineresult // array of strings. Each string is a display line of the final configuration</code></pre><p><em>e.g.</em></p><pre><code>lines = ["CAT"]rotors = [[1,13,27]]result = ["DOG"]</code></pre><h1 id="kata-task">Kata Task</h1><p>Given the initial display lines and the rotor moves for each line, determine what the board will say after it has been fully updated.</p><p>For your convenience the characters of each rotor are in the pre-loaded constant <code>ALPHABET</code> which is a string.</p><hr><p><em>And don't forget to try my other flap display Katas!</em></p><p>:-)</p>
