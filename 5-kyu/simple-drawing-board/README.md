<h2><a href=https://www.codewars.com/kata/58eb35b19925753a750000eb/train/javascript target="_blank">Simple Drawing Board</a></h2><h3>5 kyu</h3><p>You are going to write a simple console-style drawing board program.</p><h3 id="functions">Functions</h3><p>The drawing board is very simple so only these functions are supported.</p><table><thead><tr><th>Function</th><th>Description</th></tr></thead><tbody><tr><td>Create canvas</td><td>Create new empty canvas (filled with ' ') with given width and height</td></tr><tr><td>Draw line / rectangle</td><td>Draw the line with 'x' which connects the given points.<br> If the points are diagonal, a rectangle instead of diagonal line should be drawn</td></tr><tr><td>Fill color</td><td>Fill the entire area connected to the given point with given character, also known as "bucket fill" in paint programs</td></tr></tbody></table><h3 id="output">Output</h3><p>Implement the <code>drawCanvas</code> method to return the string contains the entire canvas surrounded by borders with <code>-</code> and <code>|</code>.</p><h3 id="example">Example</h3><p>Empty canvas 4x2</p><pre><code>------|    ||    |------</code></pre><p>Drawing 2 lines<code>(0, 2)</code> to <code>(4, 2)</code> and <code>(2, 0)</code> to <code>(2, 4)</code></p><pre><code>-------|  x  ||  x  ||xxxxx||  x  ||  x  |-------</code></pre><p>Drawing a rectangle <code>(1, 1)</code> to <code>(5, 4)</code> and fill with <code>o</code></p><pre><code>---------|       || xxxxx || xooox || xooox || xxxxx ||       |---------</code></pre><h3 id="error-handling">Error handling</h3><p>For any invalid input, raise an error/exception depends on the language.</p><ul><li>In <code>Java</code> should throw <code>java.lang.IllegalArgumentException</code></li><li>In <code>JavaScript</code> should throw the string <code>Out of bounds</code></li></ul><p>The data type of the arguments is <strong>always valid</strong>, no need to do type checks.</p><h3 id="remarks">Remarks</h3><ul><li>The drawing board functions should return the object itself to allow <code>method chaining</code></li><li>The coordinates are zero-based, therefore the origin is <code>(0, 0)</code>, which is at the top-left corner</li><li>The coordinates are inclusive in <code>draw</code> method</li><li>If the area is already filled with different color, <code>fill</code> method should do nothing</li></ul>
