<h2><a href=https://www.codewars.com/kata/59015f8cc842a3e7f10000a4/train/javascript target="_blank">Simple Fun #220: Simplified Array</a></h2><h3>5 kyu</h3><h2 id="task">Task</h2><p>You're given an array <code>arr</code>. Apply the following algorithm to it:</p><ol><li>find intervals of consecutive prime numbers and consecutive non-prime numbers;</li><li>replace each such interval with the sum of numbers in it;</li><li>if the resulting array is different from the initial one, return to step 1, otherwise return the result.</li></ol><h3 id="input">Input</h3><p>A non-empty integer array such that:</p><ul><li><code>-10000 ≤ arr[i] ≤ 10000</code></li><li><code>1 ≤ arr.length ≤ 1000</code></li></ul><h3 id="output">Output</h3><p>An integer array.</p><h3 id="examples">Examples</h3><p>For <code>arr = [1, 2, 3, 5, 6, 4, 2, 3]</code> the result should be <code>[21, 5]</code>:</p><pre><code>[1, 2, 3, 5, 6, 4, 2, 3] --&gt; [(1), (2 + 3 + 5), (6 + 4), (2 + 3)] --&gt; [1, 10, 10, 5][1, 10, 10, 5] --&gt; [(1 + 10 + 10), (5)] --&gt; [21, 5]</code></pre><p>For <code>arr = [-3, 4, 5, 2, 0, -10]</code> the result should be <code>[1, 7, -10]</code>:</p><pre><code>[-3, 4, 5, 2, 0, -10] --&gt; [(-3 + 4), (5 + 2), (0 + -10)] --&gt; [1, 7, -10]</code></pre>
