<h2><a href=https://www.codewars.com/kata/615da209cf564e0032b3ecc6/train/javascript target="_blank">Regex matching all positive numbers below n</a></h2><h3>5 kyu</h3><p>You are given a positive natural number <code>n</code> (which is <code>n &gt; 0</code>) and you should create a regular expression pattern which only matches the decimal representation of all positive natural numbers strictly less than <code>n</code> without leading zeros. The empty string, numbers with leading zeros, negative numbers and non-numbers should not match the regular expression compiled from your returned pattern.</p><h1 id="input">Input</h1><ul><li><code>n &gt; 0</code> natural number, <code>n</code> can be from the full possible positive range</li></ul><h1 id="output">Output</h1><ul><li>regular expression pattern as string which will be used to compile a regular expression to do the matches</li></ul><h1 id="tests">Tests</h1><p>The compiled regular expression will be tested against decimal representations of random numbers with and without leading zeros, strings including letters and the empty string and should only match for decimal representations of numbers <code>k</code> with <code>0 &lt; k &lt; n</code>.</p><p>Tests use <code>chai.assert.match()</code> which itself uses <code>RegExp.prototype.exec()</code> to do the matches.</p>
