<h2><a href=https://www.codewars.com/kata/58855acc9e1de22dff0000ef/train/javascript target="_blank">Esolang: Stick</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p>Make a custom esolang interpreter for the language Stick. Stick is a simple, stack-based esoteric programming language with only 7 commands.</p><h1 id="commands">Commands</h1><ul><li><p><code>^</code>: Pop the stack.</p></li><li><p><code>!</code>: Add new element to stack with the value of 0.</p></li><li><p><code>+</code>: Increment element. 255+1=0.</p></li><li><p><code>-</code>: Decrement element. 0-1=255.</p></li><li><p><code>*</code>: Add ascii value of <strong>top</strong> element to the output stream.</p></li><li><p><code>[</code>: Skip past <strong>next</strong> <code>]</code> if element value is 0.</p></li><li><p><code>]</code>: Jump back to the command after <strong>previous</strong> <code>[</code> if element value is nonzero.</p></li></ul><h1 id="syntax-and-other-info">Syntax and other info</h1><ul><li>You don't need to add support for nested brackets.</li><li>Non-command characters should be ignored.</li><li>Code will always have all brackets closed.</li><li>Note the highlighted <strong>next</strong> and <strong>previous</strong> in the commands reference.</li><li>Program begins with the top element having the value of 0 and being the only element in the stack.</li><li>Program ends when command executor reaches the end.</li></ul><h1 id="examples">Examples</h1><h2 id="hello-world">Hello, World!</h2><pre><code
