<h2><a href=https://www.codewars.com/kata/688a614adfe03af512d4458c/train/javascript target="_blank">Subset Sum</a></h2><h3>5 kyu</h3><p>Given a possibly empty list of strictly positive numbers and a non-negative target number, return either a subset of the list summing to the target, or <code>null</code> or a similar empty value if no such subset exists.</p><p>The subset must consist of unique ( by index ) list elements.<br>If a particular value occurs more than once in the input list, you can use it up to as many times as it occurs.<br>The empty subset sums to <code>0</code>.<br>If multiple valid subsets exist, return any one of them.  </p><p>The target will never be much bigger than the sum of the input list, and often quite a bit smaller.</p>
