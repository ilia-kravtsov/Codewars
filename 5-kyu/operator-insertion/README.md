<h2><a href=https://www.codewars.com/kata/596b7f284f232df61e00001b/train/javascript target="_blank">Operator Insertion</a></h2><h3>5 kyu</h3><p>An expression is formed by taking the digits 1 to 9 in numerical order and then inserting into each gap between the numbers either a plus sign or a minus sign or neither.</p><p>Your task is to write a method which takes one parameter and returns the <strong>smallest possible number</strong> of plus and minus signs necessary to form such an expression which equals the input.</p><p><strong>Note:</strong> All digits from 1-9 must be used exactly once.</p><p>If there is no possible expression that evaluates to the input, then return <code>null/nil/None</code>.</p><p>There are 50 random tests (100 in JavaScript), with input being an integer with with upper bound of 1000.</p><h2 id="examples">Examples</h2><p>When the input is 100, you need to return <code>3</code>, since that is the minimum number of signs required, because: 123 - 45 - 67 + 89 = 100 (3 operators in total).</p><p>More examples:</p><pre><code> 11  --&gt;  5  #  1 + 2 + 34 + 56 + 7 - 89 = 11100  --&gt;  3  #  123 - 45 - 67 + 89 = 100766  --&gt;  4  #  1 - 2 + 34 - 56 + 789 = 766160  --&gt;  -  #  no solution possible</code></pre><p>Inspired by a <a href="https://www.bbc.co.uk/programmes/p057wxwl" data-turbolinks="false" target="_blank">puzzle on BBC Radio 4</a> (which is unfortunately not available anymore)</p>
