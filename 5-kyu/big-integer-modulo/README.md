<h2><a href=https://www.codewars.com/kata/546e0773fa8da2013200087a/train/javascript target="_blank">Big Integer Modulo</a></h2><h3>5 kyu</h3><p>Calculating the remainder (modulo) of a given division is a pretty common task in programming as you might already know. Applications include checksum verification, prime numbers testing and cryptography, among others.</p><p>The default integer modulo implementations (<code>%</code>) for many languages only deal limited-length integers (like 32-bit or 64-bit integers); thus, they can't be used for calculating the modulo of greater or arbitrary-length integers.</p><p>Your task for this kata is to implement the function <code>bigModulo(numString, divisor)</code> where <code>numString</code> is a <strong>string</strong> containg the integer to divide and <code>divisor</code> is the <strong>integer</strong> by which you're going to divide.</p><p>For the tests, you can assume that:</p><ul><li><code>numString</code> has an arbitrary-length string containing a positive (<code>&gt;0</code>) integer</li><li><code>divisor</code> will always be a native integer greater than 1.</li></ul><p>You cannot use <code>bignumber.js</code> in JavaScript and your total code length should be less than or equal to 3000 symbols.</p><h3 id="hints">Hints</h3><ul><li>Remember that decimal integers can be represented as a sum of products of its digits by powers of 10.</li><li>There are certain mathematical properties of modulos regarding to sums and multiplications, you should check them if you don't know (or remember) them.</li></ul>
