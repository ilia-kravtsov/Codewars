<h2><a href=https://www.codewars.com/kata/58ad0e0a154165a1c80000ea/train/javascript target="_blank">Simple Fun #155: Pac-Man</a></h2><h3>5 kyu</h3><h2 id="task">Task</h2><p> Pac-Man got lucky today! Due to minor performance issue all his enemies have frozen. Too bad Pac-Man is not brave enough to face them right now, so he doesn't want any enemy to see him. </p><p> Given a gamefield of size <code>N</code> x <code>N</code>, Pac-Man's position(<code>PM</code>) and his enemies' positions(<code>enemies</code>), your task is to count the number of coins he can collect without being seen.</p><p> An enemy can see a Pac-Man if they are standing on the same row or column. </p><p> It is guaranteed that no enemy can see Pac-Man on the starting position. There is a coin on each empty square (i.e. where there is no Pac-Man or enemy).</p><h2 id="example">Example</h2><p> For <code>N = 4, PM = [3, 0], enemies = [[1, 2]]</code>, the result should be <code>3</code>.</p><pre><code>Let O represent coins, P - Pac-Man and E - enemy.OOOOOOEOOOOOPOOO</code></pre><p>Pac-Man cannot cross row 1 and column 2. </p><p> He can only collect coins from points <code>(2, 0), (2, 1) and (3, 1)</code>, like this:</p><pre><code>x is the points that Pac-Man can collect the coins.OOOOOOEOxxOOPxOO</code></pre><h2 id="inputoutput">Input/Output</h2><ul><li><code>[input]</code> integer <code>N</code></li></ul><p>  The field size.</p><ul><li><code>[input]</code> integer array <code>PM</code></li></ul><p>  Pac-Man's position (pair of integers)</p><ul><li><code>[input]</code> 2D integer array <code>enemies</code></li></ul><p>  Enemies' positions (array of pairs)</p><ul><li><code>[output]</code> an integer</li></ul><p>  Number of coins Pac-Man can collect.</p><h1 id="more-pacman-katas">More PacMan Katas</h1><ul><li><p><a href="https://www.codewars.com/kata/575c29d5fcee86cb8b000136" data-turbolinks="false" target="_blank">Play PacMan: Devour all</a></p></li><li><p><a href="https://www.codewars.com/kata/575ed46e23891f67d90000d8" data-turbolinks="false" target="_blank">Play PacMan 2: The way home</a></p></li></ul>
