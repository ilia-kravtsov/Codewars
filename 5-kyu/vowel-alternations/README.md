<h2><a href=https://www.codewars.com/kata/54221c16dda52609b1000ffb/train/javascript target="_blank">Vowel Alternations</a></h2><h3>5 kyu</h3><p>Write a program that performs a search for solutions to a word game.  The goal of the game is to find sets of five words that share a vowel alternation.  E.g., one solution to the game might be the words:</p><pre><code>['last', 'lest', 'list', 'lost', 'lust']</code></pre><p>This list above is a valid solution instance since the vowels <code>'a'</code>, <code>'e'</code>, <code>'i'</code>, <code>'o'</code>, and <code>'u'</code> occur in the same positions between the consonant clusters <code>'l'</code> and <code>'st'</code>.</p><p>Your program should return a list of all solutions (like the list above) given a list of words to search through.</p><p>Note that <strong>order matters</strong>.  If you prefer to use sets for book-keeping purposes, that is fine, but you will need to sort your list of solutions, and each solution set itself.</p><p>This is easy in Python:</p><pre><code>solutions = [set(['last', 'lest', 'list', 'lost', 'lust']),             set(['bill', 'bull', 'bell', 'ball', 'boll'])]sorted_soultions = sorted(sorted(solution) for solution in solutions)sorted_solutions == [['ball', 'bell', 'bill', 'boll', 'bull'],                     ['last', 'lest', 'list', 'lost', 'lust']]</code></pre><p>For the sake of simplificiation, you may assume that the words in the list you are searching over are unique.</p>
