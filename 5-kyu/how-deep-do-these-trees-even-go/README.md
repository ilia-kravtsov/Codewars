You need to quantify how deep these arrays go by calculating these numbers:</p><ul><li>min: minimum depth</li><li>max: maximum depth</li><li>mean: average depth</li></ul><p>and return the result as an object:</p><pre><code>{  min: &lt;some number&gt;,  max: &lt;some number&gt;,  mean: &lt;some number&gt;}</code></pre><p>Depth is quantified by all the leaves (i.e elements that are not arrays) of the tree.</p><p>It is guaranteed that the initial array will always be an array, and there will never be any circular references.</p>
