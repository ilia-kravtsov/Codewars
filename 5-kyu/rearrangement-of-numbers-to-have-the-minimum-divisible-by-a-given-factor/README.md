<h2><a href=https://www.codewars.com/kata/569e8353166da6908500003d/train/javascript target="_blank">Rearrangement of Numbers to Have The Minimum  Divisible by a Given Factor</a></h2><h3>5 kyu</h3><p>A function receives a certain numbers of integers <code>n1, n2, n3 ..., np</code>(all positive and different from 0) and a factor <code>k, k &gt; 0</code></p><p>The function rearranges the numbers <code>n1, n2, ..., np</code> in such order that generates the minimum number concatenating the digits and this number should be divisible by <code>k</code>.</p><p>The order that the function receives their arguments is:</p><pre><code class="language-python"><span class="cm-variable">rearranger</span>(<span class="cm-variable">k</span>, <span class="cm-variable">n1</span>, <span class="cm-variable">n2</span>, <span class="cm-variable">n3</span>,<span class="cm-operator">...</span>.,<span class="cm-variable">np</span>)</code></pre><h2 id="examples-input----output">Examples (input --&gt; output)</h2><pre><code>rearranger(4, 32, 3, 34, 7, 12)  --&gt; "Rearrangement: 12, 3, 34, 7, 32 generates: 12334732 divisible by 4"rearranger(10, 32, 3, 34, 7, 12) --&gt; "There is no possible rearrangement"</code></pre><p>If there are more than one possible arrangements for the same minimum number, your code should be able to handle those cases:</p><pre><code>6, [19, 32, 2, 124, 20, 22] --&gt; "Rearrangements: 124, 19, 20, 2, 22, 32 and 124, 19, 20, 22, 2, 32 generate: 124192022232 divisible by 6"</code></pre><p>The arrangements should be in sorted order, as you see: <code>124, 19, 20, 2, 22, 32</code> comes first than <code>124, 19, 20, 22, 2, 32</code>.</p><p>Have an enjoyable time!</p><p>(Thanks to <code>ChristianE.Cooper</code> for his contribution to this kata)</p>
