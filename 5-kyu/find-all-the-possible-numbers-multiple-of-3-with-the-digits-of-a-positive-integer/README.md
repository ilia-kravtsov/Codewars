<h2><a href=https://www.codewars.com/kata/5828b9455421a4a4e8000007/train/javascript target="_blank">Find All the Possible Numbers Multiple of 3 with the Digits of a Positive Integer.</a></h2><h3>5 kyu</h3><p>Given a certain number, how many multiples of three could you obtain with its digits?</p><p>Suposse that you have the number 362. The numbers that can be generated from it are:</p><pre><code>362 ----&gt; 3, 6, 2, 36, 63, 62, 26, 32, 23, 236, 263, 326, 362, 623, 632 </code></pre><p>But only <code>3, 6, 36, 63</code> are multiple of three.</p><p>We need a function that can receive a number and may output in the following order:</p><ul><li><p>the amount of multiples</p></li><li><p>the maximum multiple</p></li></ul><p>Let's see a case the number has a the digit 0 and repeated digits:</p><pre><code>6063 ----&gt; 0, 3, 6, 30, 36, 60, 63, 66, 306, 360, 366, 603, 606, 630, 636, 660, 663, 3066, 3606, 3660, 6036, 6063, 6306, 6360, 6603, 6630</code></pre><p>In this case the multiples of three will be all except 0</p><pre><code>6063 ----&gt; 3, 6, 30, 36, 60, 63, 66, 306, 360, 366, 603, 606, 630, 636, 660, 663, 3066, 3606, 3660, 6036, 6063, 6306, 6360, 6603, 6630</code></pre><p>So the expected result given the above cases:</p><pre><code>Given 362, should return [4, 63](amount of multiples = 4, maximum multiple = 63)Given 6063, should return [25, 6630](amount of multiples = 25, maximum multiple = 6630)</code></pre><p>The function will receive only positive integers (num &gt; 0), and you don't have to worry for validating the entries.</p><p>Features of the random tests:</p><pre><code>Number of test = 1001000 ≤ num ≤ 100000000</code></pre><p>Enjoy it!!</p>
