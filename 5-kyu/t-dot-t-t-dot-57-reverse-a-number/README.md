<h2><a href=https://www.codewars.com/kata/57c786e858da9e5ed20000ea/train/javascript target="_blank">T.T.T.57: Reverse a number</a></h2><h3>5 kyu</h3><h2 id="description">Description</h2><p>Given a string representing an integer, return a string with every sequence of ascending or descending digits reversed.</p><p>Examples:</p><pre><code>123456  ---&gt;  654321  // ascending sequence 123456654321  ---&gt;  123456  // descending sequence 654321 135246  ---&gt;  531642  // ascending sequence 135, ascending sequence 246642531  ---&gt;  246135  // descending sequence 642, descending sequence 531</code></pre><p>Such sequences are not required to be strictly ascending / descending:</p><pre><code>12333  ---&gt; 33321     // ascending sequence 1233366654  ---&gt; 45666     // descending sequence 66654133555224466  ---&gt; 555331664422   // ascending sequence 133555, ascending sequence 224466 </code></pre><p>If a digit (or several of the same consecutive digits) is at the junction part of ascending and descending sequence, it should belong to the left part of the junction:</p><pre><code>12321  ---&gt; 32112  digit 3 may be part of an ascending order(123)It can also be a part of a descending order(321)135642  ---&gt;  653124digit 6 may be part of an ascending order(1356)It can also be a part of a descending order(642)13555432  ---&gt; 55531234consecutive digits 555 may be part of an ascending order(13555)It can also be a part of a descending order(555432)</code></pre><p>If the number is a negative number, the symbol <code>-</code> does not reverse:</p><pre><code>-123  ---&gt; -321-123321  ---&gt;  -332112</code></pre><p>After reverse, if the digit 0 is in the first place (ignoring the sign), it should be removed:</p><pre><code>420135  ---&gt; 024531  ---&gt; 24531-520025  ---&gt; -002552  ---&gt;  -2552</code></pre><h2 id="task">Task</h2><p>Complete function <code>reverseNumber()</code> that accepts a arguments <code>n</code>(given by string format).</p><p>Returns the number that reverse in accordance with the rules(string format).</p>
