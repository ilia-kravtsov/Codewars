<h2><a href=https://www.codewars.com/kata/583353eab189a5b860000049/train/javascript target="_blank">Lazy Chain</a></h2><h3>5 kyu</h3><p>Method chaining is a useful technique in JavaScript, allowing better composability and readability of functions.</p><p>One problem with standard JavaScript chaining is that chained methods are immediately executed upon creation.</p><pre><code class="language-javascript">  <span class="cm-keyword">let</span> <span class="cm-def">a</span> <span class="cm-operator">=</span> [ <span class="cm-number">1</span>, <span class="cm-number">2</span>, <span class="cm-number">3</span>].<span class="cm-property">map</span>(<span class="cm-def">x</span> <span class="cm-operator">=&gt;</span> <span class="cm-variable-2">x</span> <span class="cm-operator">*</span> <span class="cm-number">2</span>).<span class="cm-property">reverse</span>();  <span class="cm-comment">// a = [ 6, 4, 2]</span></code></pre><p>The problem with this is that it opens values up to possible mutation before being called.</p><pre><code class="language-javascript">  <span class="cm-variable">a</span>.<span class="cm-property">join</span>(<span class="cm-string">""</span>);  <span class="cm-comment">// a = "642"</span>    <span class="cm-keyword">function</span> <span class="cm-def">doSomething</span>(<span class="cm-def">array</span>) {    <span class="cm-comment">// Does something to an array</span>  }    <span class="cm-variable">doSomething</span>(<span class="cm-variable">a</span>); <span class="cm-comment">// TypeError - a is not an array  </span></code></pre><p>To combat this we can create a function lazyChain that accepts an argument and allows chaining, but delays execution to when the value method is called.</p><pre><code class="language-javascript">  <span class="cm-keyword">let</span> <span class="cm-def">a</span> <span class="cm-operator">=</span> <span class="cm-variable">lazyChain</span>([<span class="cm-number">1</span>,<span class="cm-number">2</span>,<span class="cm-number">3</span>])                            .<span class="cm-property">invoke</span>(<span class="cm-string">'map'</span>, <span class="cm-def">x</span> <span class="cm-operator">=&gt;</span> <span class="cm-variable-2">x</span> <span class="cm-operator">*</span> <span class="cm-number">2</span>)                            .<span class="cm-property">invoke</span>(<span class="cm-string">'reverse'</span>);                            <span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">a</span>.<span class="cm-property">value</span>());  <span class="cm-comment">// [ 6, 4, 2]</span>                              <span class="cm-variable">a</span>.<span class="cm-property">join</span>(<span class="cm-string">""</span>);  <span class="cm-comment">// TypeError</span>    <span class="cm-variable">doSomething</span>(<span class="cm-variable">a</span>.<span class="cm-property">value</span>());  <span class="cm-comment">// valid function call</span></code></pre><p>Your task is to create the function lazyChain which accepts any value, and allows method chaining through the use of invoke, and execution with the value method.</p><p>Keep in mind that invoke is general enough to accept any prototype methods from the standard language.</p>
