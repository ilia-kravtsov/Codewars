<h2><a href=https://www.codewars.com/kata/52de672aa5bde77f360003bf/train/javascript target="_blank">The good old olden early years</a></h2><h3>5 kyu</h3><p>Write a function which returns the day of the week for a specified date.</p><p>While this sounds straight forward the years this function will be testing against are only within the range of 1-99.  </p><p>Any date range past 99 or before 1 will not have unit tests against.</p><p>All dates in the unit tests used will be in the format of DD-MM-YYYY.</p><p>17th of February 5 will be passed as <code>'17-02-0005'</code>, though <code>'17-2-5'</code> should work just the same.</p><p>Example:</p><pre><code>getGoodOldDay('19-11-0017'); // Returns SundaygetGoodOldDay('5-7-53'); // Returns Saturday</code></pre><p>Note/Hint:  </p><blockquote><p>If in doubt, check the full date you are about to return the day of the week for.</p></blockquote>
