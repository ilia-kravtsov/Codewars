<h2><a href=https://www.codewars.com/kata/638b042bf418c453377f28ad/train/javascript target="_blank">Digits</a></h2><h3>5 kyu</h3><p>The code consists of four unique digits (from <code>0</code> to <code>9</code>).<br>Tests will call your solution; you should answer with an array of four digits.<br>Your input is number of matches (the same digit in the same place) with your previous answer. For the first call input value is <code>-1</code> (i.e. each new test starts with input <code>-1</code>)<br>You have to find the code in <code>16</code> calls or less. You are the best. Do it.</p><h2 id="for-example">For example</h2><p>The code is <code>[1, 2, 3, 4]</code><br>1st call <code>return [1, 3, 4, 5]</code> will give <code>1</code> match in next input<br>2nd call <code>return [1, 2, 3, 0]</code> will give <code>3</code> matches in next input<br>3rd call <code>return [1, 2, 3, 4]</code> will not give <code>4</code> matches in next input, because you're the champion!</p><h2 id="ps">P.S.</h2><p>If the task seems too difficult, try <a href="https://www.goobix.com/games/guess-the-number/" data-turbolinks="false" target="_blank">this game</a> with almost the same rules first.</p>
