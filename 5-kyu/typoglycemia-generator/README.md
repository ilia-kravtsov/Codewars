<h2><a href=https://www.codewars.com/kata/55953e906851cf2441000032/train/javascript target="_blank">Typoglycemia Generator</a></h2><h3>5 kyu</h3><p><b>Background</b><br>There is a message that is circulating via public media that claims a reader can easily read a message where the inner letters of each words is scrambled, as long as the first and last letters remain the same and the word contains all the letters.</p><p>Another example shows that it is quite difficult to read the text where all the letters are reversed rather than scrambled.</p><p>In this kata we will make a generator that generates text in a similar pattern, but instead of scrambled or reversed, ours will be sorted alphabetically</p><p><b>Requirement</b><br>return a string where:<br></p><ol><li>the first and last characters remain in original place for each word<br></li><li>characters between the first and last characters must be sorted alphabetically<br></li><li>punctuation should remain at the same place as it started, for example: shan't -&gt; sahn't<br></li></ol><p><b>Assumptions</b><br></p><ol><li>words are seperated by single spaces<br></li><li>only spaces separate words, special characters do not,  for example: tik-tak -&gt; tai-ktk<br></li><li>special characters do not take the position of the non special characters, for example: -dcba  -&gt;  -dbca<br></li><li>for this kata puctuation is limited to 4 characters: hyphen(-), apostrophe('), comma(,) and period(.) <br></li><li>ignore capitalisation<br></li></ol><p>for reference: <a href="http://en.wikipedia.org/wiki/Typoglycemia" data-turbolinks="false" target="_blank">http://en.wikipedia.org/wiki/Typoglycemia</a></p>
