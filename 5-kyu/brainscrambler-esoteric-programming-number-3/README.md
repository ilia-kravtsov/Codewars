id="brainscrambler">Brainscrambler</h1><p><a href="https://esolangs.org/wiki/Brainscrambler" data-turbolinks="false" target="_blank">Brainscrambler</a> is an Esoteric programming language.</p><p>A Brainscrambler program consists of a string containing</p><ul><li>command characters</li><li>decimal numbers, which are the input of the program.</li></ul><p>Brainscrabler memory is made up of three stacks storing integers :<code>A</code>, <code>B</code>, and <code>C</code>.The "current stack" starts out at <code>A</code>, then is rotated to <code>B</code>, then to <code>C</code>, then back to <code>A</code> again, etc.The "current value" is the number at the top of the current stack.</p><p>In other words :</p><ul><li><code>A</code> is to the right of <code>C</code> and to the left of <code>B</code></li><li><code>B</code> is to the right of <code>A</code> and to the left of <code>C</code></li><li><code>C</code> is to the right of <code>B</code> and to the left of <code>A</code></li></ul><p>Your task is to create an interpreter for Brainscrambler.You have to write the method <code>read(input)</code> of the object <code>Interpreter</code>.</p><p>This method takes as parameter  a <code>string</code>  representing the source code of the Brainscrambler program.</p><p>Its output shall be a string containing all the values that have been sent to the output during the execution of the program.</p><p>You must execute the commands until you reach the end of the input string.</p><h2 id="commands">Commands</h2><p>The eleven possible commands in the Brainscrambler language:</p><ul><li><code>+</code> Increment the current number.</li><li><code>-</code> Decrement the current number.</li><li><code>&lt;</code> Move the current number to the stack to the "left".</li><li><code>&gt;</code> Move the current number to the stack to the "right".</li><li><code>*</code> Push a zero onto the current stack.</li><li><code>^</code> Pop the current number and discard it.</li><li><code>#</code> Rotate between stacks.</li><li><code>,</code> Input a number and push it to the current stack. This number will consist of the decimal digits immediatly to the right of the <code>,</code> character. After reading the number, the program resumes at the command to the right of the last digit.</li><li><code>.</code> Output the current number.</li><li><code>[</code> Enter the loop.</li><li><code>]</code> Jump to the corresponding <code>[</code> bracket if the current number is &gt; <code>0</code>.</li></ul><h2 id="specifications">Specifications</h2><ul><li><p>All stacks have an original value of 0</p></li><li><p>The input numbers are decimal and in the big-endian order.</p></li><li><p>When popping from the stack, if the stack is empty, the current number will be undefined. If the current number is undefined :</p><ul><li>If a <code>.</code> or <code>&lt;</code> or <code>&gt;</code> operation appears, do not add anything to the output or shift anything.</li><li>If a <code>+</code> or <code>-</code> appears, set the current number to <code>0</code>.</li><li>If a <code>]</code> appears, do not reenter the loop (as <code>undefined &gt; 0 === false</code>).</li></ul></li><li><p>Between <code>read()</code> calls :</p><ul><li>The output is reset;</li><li>The stacks keep their values;</li><li>The current stack stays the same.</li></ul></li><li><p>The loops are <code>do [] while</code> loops, i.e. the loop body is executed BEFORE evaluating the current value</p></li><li><p>Loops can be nested</p></li></ul><h2 id="examples">Examples</h2><h4 id="simple-program">Simple Program</h4><pre><code class="language-javascript"><span class="cm-keyword">const</span> <span class="cm-def">interpreter</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">Interpreter</span>();<span class="cm-variable">interpreter</span>.<span class="cm-property">read</span>(<span class="cm-string">'*+.'</span>);<span class="cm-string">'1'</span></code></pre><pre style="display: none;"><code class="language-python"><span class="cm-variable">interpreter</span> <span class="cm-operator">=</span> <span class="cm-variable">Interpreter</span>()<span class="cm-variable">interpreter</span>.<span class="cm-property">read</span>(<span class="cm-string">'*+.'</span>)<span class="cm-string">'1'</span></code></pre>
