<h2><a href=https://www.codewars.com/kata/578e07d590f2bb8d3300001d/train/javascript target="_blank">WordSquare</a></h2><h3>5 kyu</h3><p>A <a href="https://en.wikipedia.org/wiki/Word_square" data-turbolinks="false" target="_blank">Word Square</a> is a set of words written out in a square grid, such that the same words can be read both horizontally and vertically. The number of words, equal to the number of letters in each word, is known as the <em>order</em> of the square.</p><p>For example, this is an <em>order</em> <code>5</code> square found in the ruins of Herculaneum:</p><p><img alt="" src="https://i.gyazo.com/e226262e3ada421d4323369fb6cf66a6.jpg"></p><p>Given a string of various uppercase <code>letters</code>, check whether a <em>Word Square</em> can be formed from it. </p><p>Note that you should use each letter from <code>letters</code> the exact number of times it occurs in the string. If a <em>Word Square</em> can be formed, return <code>true</code>, otherwise return <code>false</code>.</p><p><strong>Example</strong></p><ul><li><p>For <code>letters = "SATORAREPOTENETOPERAROTAS"</code>, the output should be  <code>WordSquare(letters) = true</code>.It is possible to form a <em>word square</em> in the example above.</p></li><li><p>For <code>letters = "AAAAEEEENOOOOPPRRRRSSTTTT"</code>, (which is sorted form of <code>"SATORAREPOTENETOPERAROTAS"</code>), the output should also be  <code>WordSquare(letters) = true</code>.</p></li><li><p>For <code>letters = "NOTSQUARE"</code>, the output should be  <code>WordSquare(letters) = false</code>.</p></li></ul><p><strong>Input/Output</strong></p><ul><li><p>[input] string letters</p><p>A string of uppercase English letters.</p><p>Constraints: <code>3 ≤ letters.length ≤ 100</code>.</p></li><li><p>[output] boolean</p><p><code>true</code>, if a Word Square can be formed;</p><p><code>false</code>, if a Word Square cannot be formed.</p></li></ul>
