<h2><a href=https://www.codewars.com/kata/58a3e2978bdda5a0d9000187/train/javascript target="_blank">Simple Fun #127: Prime Operations</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p> Given two numbers <code>x</code> and <code>y</code>. You have to convert <code>x</code> to <code>y</code> using minimum number of operations.</p><p> There are two valid operation on the number <code>x</code>:</p><pre><code>Multiply x by some prime pDivide x by some prime p</code></pre><p>Your task is to find the minimum number of operations.</p><p>Note that integer division is NOT allowed. That is, the result of <code>7 / 2</code> is 3.5, instead of 3.</p><h1 id="example">Example</h1><p> For: <code>x = 2, y = 3</code>, the result should be <code>2</code>.</p><p> <code>2 -&gt; 2 * 3 -&gt; 6 / 2 -&gt; 3</code> in 2 moves.</p><h1 id="inputoutput">Input/Output</h1><ul><li><code>[input]</code> integer <code>x</code></li></ul><p>  A decimal positive integer. <code>0 &lt; x &lt;= 10^12</code></p><ul><li><code>[input]</code> integer <code>y</code></li></ul><p>  A decimal positive integer. <code>0 &lt; y &lt;= 10^12</code></p><ul><li><code>[output]</code> an integer</li></ul><p>  The minimum number of operations</p>
