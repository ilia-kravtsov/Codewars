<h2><a href=https://www.codewars.com/kata/57b06eb0bb9944d42c0001ab/train/javascript target="_blank">T.T.T.33: Grandfather clock</a></h2><h3>5 kyu</h3><h2 id="description">Description</h2><p>There is a grandfather clock. At one o'clock, it struck once; At two o'clock, it struck 2 times; etc.. At twelve o'clock, it struck 12 times. In addition, in half an hour, it struck once.</p><p>Give you a time period, can you calculate how many times it strucks?</p><h2 id="task">Task</h2><p>Complete function <code>howManyTimes()</code> that accepts two argument <code>time1</code> and <code>time2</code>, they are two strings like this:<code>"yyyy-MM-dd hh:mm:ss"</code></p><p>You need to calculate how many times it strucks and return an integer value.</p><h3 id="specification">Specification</h3><ul><li>Each struck takes 1 second.</li><li><code>time1</code> is always earlier than <code>time2</code>. </li><li><code>time1</code> is inclusive, <code>time2</code> is exclusive.</li><li>Strucks use the 12-hour system.
