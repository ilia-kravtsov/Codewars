<h2><a href=https://www.codewars.com/kata/5a3267b2ee1aaead3d000037/train/javascript target="_blank">1-800-CODE-WAR</a></h2><h3>5 kyu</h3><img src="https://i.imgur.com/ta6gv1i.png?1"><hr><h1 id="background">Background</h1><p>If your phone buttons have letters, then it is easy remember long phone numbers by making words from the substituted digits. </p><p><a href="https://en.wikipedia.org/wiki/Phoneword" data-turbolinks="false" target="_blank">https://en.wikipedia.org/wiki/Phoneword</a></p><img title="source: imgur.com" src="https://i.imgur.com/VJU55cg.png"><p>This is common for 1-800 numbers</p><h1 id="1-800-number-format">1-800 number format</h1><p>This format probably varies for different countries, but for the purpose of this Kata here are <strong>my</strong> rules:</p><p>A 1-800 number is an 11 digit phone number which starts with a <code>1-800</code> prefix.</p><p>The remaining 7 digits can be written as a combination of 3 or 4 letter words. e.g.</p><ul><li><code>1-800-CODE-WAR</code></li><li><code>1-800-NEW-KATA</code></li><li><code>1-800-GOOD-JOB</code></li></ul><p>The <code>-</code> are included just to improve the readability.</p><h1 id="story">Story</h1><p>A local company has decided they want to reserve a 1-800 number to help with advertising. </p><p>They have already chosen what words they want to use, but they are worried that maybe that same phone number could spell out other words as well. Maybe bad words. Maybe embarrassing words. </p><p>They need somebody to check for them so they can avoid any accidental PR scandals!</p><p>That's where you come in...</p><h1 id="kata-task">Kata Task</h1><p>There is a preloaded array of lots of 3 and 4 letter (upper-case) words:</p><p><code>Preloaded.WORDS</code></p><p>Given the 1-800 (phone word) number that the company wants to use, you need to check against all known words and return the set of all possible 1-800 numbers which can be formed using those same digits.</p><h1 id="notes">Notes</h1><ul><li>The desired phone-word number provided by the company is formatted properly. No need to check. </li><li>All the letters are upper-case. No need to check.</li><li>Only words in the list are allowed.</li></ul><hr><div style="color:orange">Good luck!<br>DM</div>
