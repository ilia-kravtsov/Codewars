<h2><a href=https://www.codewars.com/kata/59cf0ba5d751dffef300001f/train/javascript target="_blank">Simple Fun #358: Vertical Histogram Of Letters</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p>You are given a string <code>s</code>. Your task is to count the number of each letter (A-Z), and make a vertical histogram as result. Look at the following examples to understand the rules.</p><h1 id="example">Example</h1><p>For <code>s = "XXY YY ZZZ123ZZZ AAA BB C"</code>, the output should be:</p><pre><code>          *          *          **       * ** *   * * ** * * * * *A B C X Y Z</code></pre><h1 id="rules">Rules</h1><ul><li>You just need to count the uppercase letters. Any other character will be ignored.</li><li>Using <code>*</code> to represent the number of characters. </li><li>The order of output is form A to Z. Characters that do not appear in the string are ignored.</li><li>To beautify the histogram, there is a space between every pair of columns.</li><li>There are no extra spaces at the end of each row. Also, use "\n" to separate rows.</li><li>Happy Coding <code>^_^</code></li></ul>
