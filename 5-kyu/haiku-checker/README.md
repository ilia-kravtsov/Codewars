<h2><a href=https://www.codewars.com/kata/5c765a4f29e50e391e1414d4/train/javascript target="_blank">Haiku Checker</a></h2><h3>5 kyu</h3><p><a href="https://en.wikipedia.org/wiki/Haiku_in_English" data-turbolinks="false" target="_blank">Haikus</a> are short poems in a three-line format, with 17 syllables arranged in a 5–7–5 pattern. Your task is to check if the supplied text is a haiku or not.</p><h4 id="about-syllables">About syllables</h4><p><a href="https://en.wikipedia.org/wiki/Syllable" data-turbolinks="false" target="_blank">Syllables</a> are the phonological building blocks of words. <em>In this kata</em>, a syllable is a part of a word including a vowel ("a-e-i-o-u-y") or a group of vowels (e.g. "ou", "ee", "ay"). A few examples: <em>"tea", "can", "to·day", "week·end", "el·e·phant".</em></p><p><strong>However</strong>, silent "E"s <strong>do not</strong> create syllables. <em>In this kata</em>, an "E" is considered silent if it's alone at the end of the word, preceded by one (or more) consonant(s) and there is at least one other syllable in the word. Examples: <em>"age", "ar·range", "con·crete";</em> but not in <em>"she", "blue", "de·gree".</em></p><p>Some more examples:</p><ul><li>one syllable words: <em>"cat", "cool", "sprout", "like", "eye", "squeeze"</em></li><li>two syllables words: <em>"ac·count", "hon·est", "beau·ty", "a·live", "be·cause", "re·store"</em></li></ul><h3 id="examples">Examples</h3><pre><code>An old silent pond...A frog jumps into the pond,splash! Silence again.</code></pre><p>...should return <code>True</code>, as this is a valid 5–7–5 haiku:</p><pre><code>An old si·lent pond...            # 5 syllablesA frog jumps in·to the pond,      # 7splash! Si·lence a·gain.          # 5</code></pre><p>Another example:</p><pre><code>Autumn moonlight -a worm digs silentlyinto the chestnut.</code></pre><p>...should return <code>False</code>, because the number of syllables per line is not correct:</p><pre><code>Au·tumn moon·light -          # 4 syllablesa worm digs si·lent·ly        # 6in·to the chest·nut.          # 5</code></pre><hr><h3 id="my-other-katas">My other katas</h3><p>If you enjoyed this kata then please try <a href="https://www.codewars.com/users/anter69/authored" data-turbolinks="false" target="_blank">my other katas</a>! :-)</p><h4 id="translations-are-welcome"><em>Translations are welcome!</em></h4>
