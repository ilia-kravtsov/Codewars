For example:</p><pre><code>arr1=[1,2,3,4,5]arr2=[9,8,0,0,0]findPair(arr1,arr2) should return [[1,9],[2,8]]because:1  2  3 4 5+  +  + + +9  8  0 0 0|  |  | | |10 10 3 4 51,9 and 2,8 have the same sum value 10</code></pre><p> You do not need to verify the input data. arr1 and arr2 will always have the same number of elements, and testcase will not provide an empty array.</p><p> If all pairs has diffrent sum value, please return <code>[]</code>.</p><p> If more than one groups of pairs has same number, choose which has the max sum value(see the last example)</p><p> You don't need sort the result, return the pairs according to the order of the index.</p><h1 id="examples">Examples</h1><pre><code>arr1=[1,2,3,4,5]arr2=[0,0,0,0,0]findPair(arr1,arr2) should return []arr1=[1,2,3,4,5]arr2=[5,4,3,2,1]findPair(arr1,arr2) should return [[1,5],[2,4],[3,3],[4,2],[5,1]]arr1=[0,1,3,4,5]arr2=[1,0,3,2,1]findPair(arr1,arr2) should return [[3,3],[4,2],[5,1]]arr1=[1,2,3,4,5]arr2=[-1,2,-3,4,-5]findPair(arr1,arr2) should return [[1,-1],[3,-3],[5,-5]]arr1=[1,2,3,0,5,-2]arr2=[-1,2,-3,4,-5,6]findPair(arr1,arr2) should return [[2,2],[0,4],[-2,6]]</code></pre>
