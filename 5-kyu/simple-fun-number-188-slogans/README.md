<h2><a href=https://www.codewars.com/kata/58bf79fdc8bd4432d6000029/train/javascript target="_blank">Simple Fun #188: Slogans</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p> You know the slogan <code>p</code>, which the agitators have been chanting for quite a while now. Roka has heard this slogan a few times, but he missed almost all of them and grasped only their endings. You know the string <code>r</code> that Roka has heard. </p><p> You need to determine what is the <code>minimal number</code> of times agitators repeated the slogan <code>p</code>, such that Roka could hear <code>r</code>.</p><p> It is guaranteed the Roka heard nothing but the endings of the slogan P repeated several times.</p><h1 id="example">Example</h1><p> For <code>p = "glorytoukraine", r = "ukraineaineaine"</code>, the output should be <code>3</code>.</p><p> The slogan was <code>"glorytoukraine"</code>, and Roka heard <code>"ukraineaineaine"</code>.</p><p> He could hear it as follows: <code>"ukraine"</code> + <code>"aine"</code> + <code>"aine"</code> = <code>"ukraineaineaine"</code>.</p><h1 id="inputoutput">Input/Output</h1><ul><li><code>[input]</code> string <code>p</code></li></ul><p>  The slogan the agitators chanted, a string of lowecase Latin letters.</p><ul><li><code>[input]</code> string <code>r</code></li></ul><p>  The string of lowercase Latin letters Roka has heard.</p><ul><li><code>[output]</code> an integer</li></ul><p>  The <code>minimum number</code> of times the agitators chanted.</p>
