<h2><a href=https://www.codewars.com/kata/55653db7b0075fe1f000000a/train/javascript target="_blank">5command - Esoteric programming #1</a></h2><h3>5 kyu</h3><h1>5command</h1><p>5command is an Esoteric programming language which has 5 commands, Your task is to create an interpreter for a 5command input. The FiveCommands class is provided and a function called read which will be used.</p><h2>Commands</h2><p>The five possible commands in the 5command language:<br></p><p><strong>+</strong> <code>move the pointer right on the tape</code> <br><strong>-</strong> <code>move the pointer left on the tape</code><br><strong>^</strong> <code>increment the value on the tape where the pointer is currently</code><br><strong>v</strong> <code>decrement the value on the tape where the pointer is currently</code><br><strong>*</strong> <code>print to tape the number at the pointer (without a space);</code><br></p><p></p><h2>Important</h2><p>Each time <code>*</code> is called you must add the number at the pointer to the output variable which will then be returned as a number when the program has finished.</p><p>The tape used is unlimited so everytime the pointer is at <code>0</code> and the program asks you to minus one from the pointer or go to <code>-1</code> on the tape just expand the tape at the start.</p><h2>Input</h2><p>You should expect input only for the read function inside the Interpreter class which will take two parameters: <code>input</code> and <code>debug</code>. <br><br>Input will be a string of possible commands for example: <code>^*</code>. <br><br>Debug will be a boolean either true or false which will determine if a debug value is returned. If it is false then return an empty array.</p><h2>Output</h2><p>The output expected for this kata will be an object this object will be predefined in the solution called <code>Output</code> It takes two properties: <code>output</code> and <code>debug</code>. The output must be the result of the input as a string. The debug must then be an array representation of the tape (examples below).</p><p>Output class definition.</p><pre><code class="language-javascript"><span class="cm-keyword">class</span> <span class="cm-def">Output</span> {  <span class="cm-property">constructor</span>(<span class="cm-def">out</span>, <span class="cm-def">debug</span>){    <span class="cm-keyword">this</span>.<span class="cm-property">output</span> <span class="cm-operator">=</span> <span class="cm-variable-2">out</span>;    <span class="cm-keyword">this</span>.<span class="cm-property">debug</span> <span class="cm-operator">=</span> <span class="cm-variable-2">debug</span>;  }}</code></pre><h2>Examples</h2><h4>Simple Program</h4> <p><code>^*</code></p><p>Normal Mode should output:</p><pre><code>output: 1</code></pre><p>Debug Mode should output:</p><pre><code class="language-javascript"><span class="cm-variable">output</span>: <span class="cm-string">'1'</span><span class="cm-variable">debug</span>: [<span class="cm-string">'0-&gt;1'</span>]</code></pre><p><br><br><strong>Reference:</strong> <a href="https://esolangs.org/wiki/5command" data-turbolinks="false" target="_blank">5command</a></p><div value="https://esolangs.org/wiki/Language_list"></div>
