<h2><a href=https://www.codewars.com/kata/62275b5bf6c169002379fa65/train/javascript target="_blank">Divide and maximize</a></h2><h3>5 kyu</h3><p>Your friend has a list of <code>k</code> numbers: <code>[a1, a2, a3, ... ak]</code>.</p><p>He is allowed to do an operation which consists of three steps:</p><ol><li>select two numbers: <code>ai</code> and <code>aj</code> (<code>ai % 2 = 0</code>)</li><li>replace <code>ai</code> with <code>ai / 2</code></li><li>replace <code>aj</code> with <code>aj * 2</code></li></ol><p>Help him to find the maximum sum of list elements that is possible to achieve by using this operation (possibly multiple times).<br>Return this sum modulo <code>1_000_000_007</code>, because it can be quite big.</p><h3 id="input">Input</h3><p>List of <code>k</code> elements: <code>[a1, a2, a3, ..., ak]; k &lt; 10**4</code><br>All numbers are positive and smaller than <code>10**9</code></p><h3 id="output">Output</h3><p>Maximum sum after some operations (modulo <code>1_000_000_007</code>)</p>
