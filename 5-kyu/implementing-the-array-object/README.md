<h2><a href=https://www.codewars.com/kata/5398beaf53f51088a9000fa7/train/javascript target="_blank">Implementing the Array Object</a></h2><h3>5 kyu</h3><p>Your task will be to implement an object (class) that mimics the array object, but should not actually <em>be</em> an <code>Array</code>. You are otherwise free to choose your own implementation.</p><p>The object shall be named <code>List</code> and shall have the methods <code>push</code>, <code>pop</code>, <code>shift</code>, <code>unshift</code> and <code>filter</code> defined on its prototype. Each of these methods should behave exactly as their <code>Array</code> counterpart does. (It is acceptable for <code>filter</code> to return a <code>List</code> <em>or</em> an <code>Array</code>.)<br>The <code>List</code> constructor must take an arbitrary number of arguments as initial values for your list.<br>Furthermore, your list should have a <code>length</code> property that dynamically changes as your list changes.</p><p>Here are some helpful resources: <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/push" data-turbolinks="false" target="_blank">push</a>, <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/pop" data-turbolinks="false" target="_blank">pop</a>, <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/shift" data-turbolinks="false" target="_blank">shift</a>, <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/unshift" data-turbolinks="false" target="_blank">unshift</a>, <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter" data-turbolinks="false" target="_blank">filter</a>.</p>
