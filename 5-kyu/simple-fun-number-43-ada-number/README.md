<h2><a href=https://www.codewars.com/kata/58882bdb5edef0343400002a/train/javascript target="_blank">Simple Fun #43: Ada Number</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p> Consider two following representations of a non-negative integer:</p><p> A simple decimal integer, constructed of a non-empty sequence of digits from 0 to 9;</p><p> An integer with at least one digit in a base from 2 to 16 (inclusive), enclosed between # characters, and preceded by the base, which can only be a number between 2 and 16 in the first representation. For digits from 10 to 15 characters a, b, ..., f and A, B, ..., F are used.</p><p> Additionally, both representations may contain underscore (_) characters; they are used only as separators for improving legibility of numbers and can be ignored while processing a number.</p><p> Your task is to determine whether the given string is a valid integer representation.</p><p> Note: <code>this is how integer numbers are represented in the programming language Ada.</code></p><h1 id="example">Example</h1><p> For <code>line = "123_456_789"</code>, the output should be <code>true;</code></p><p> For <code>line = "16#123abc#"</code>, the output should be <code>true</code>;</p><p> For <code>line = "10#123abc#"</code>, the output should be <code>false</code>;</p><p> For <code>line = "10#10#123ABC#"</code>, the output should be <code>false</code>;</p><p> For <code>line = "10#0#"</code>, the output should be <code>true</code>;</p><p> For <code>line = "10##"</code>, the output should be <code>false</code>.</p><h1 id="inputoutput">Input/Output</h1><ul><li><code>[input]</code> string <code>line</code></li></ul><p>  A non-empty string.</p><ul><li><code>[output]</code> a boolean value</li></ul><p>  <code>true</code> if line is a valid integer representation, <code>false</code> otherwise.</p>
