<h2><a href=https://www.codewars.com/kata/58e2835b5e2ad23be1000044/train/javascript target="_blank">One Line Task: Integer Average</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p> Given an array of positive integers, return <code>true</code> if the average of its elements is an integer and <code>false</code> otherwise.</p><h1 id="code-limit">Code Limit</h1><p> Less than 46 characters.</p><h1 id="example">Example</h1><p> For <code>a = [1, 2, 3]</code>, the output should be <code>true</code>.</p><p> <code>(1 + 2 + 3) / 3 = 2</code></p><p> For <code>a = [1, 2, 2]</code>, the output should be <code>false</code>.</p><p> <code>(1 + 2 + 2) / 3 = 1.666666667</code></p><p> For <code>a = [1, 9, 8, 2]</code>, the output should be <code>true</code>.</p><p> <code>(1 + 9 + 8 + 2) / 4 = 5</code></p>
