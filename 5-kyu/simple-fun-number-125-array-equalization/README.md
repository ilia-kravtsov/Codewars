<h2><a href=https://www.codewars.com/kata/58a3c836623e8c72eb000188/train/javascript target="_blank">Simple Fun #125: Array Equalization</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p>You are given an array <code>a</code> of positive integers and an intger <code>k</code>.
