The length <code>lg</code> of this array will be a positive multiple of 4.</p><p>Let </p><p><code>P = (x(1) ** 2 + x(2) ** 2) * (x(3) ** 2 + x(4) ** 2) * ... * (x(m) ** 2 + x(m+1) ** 2)</code>,</p><p><code>x ** y</code> means x raised to the power y.</p><h4 id="task">Task</h4><p>Given an array or list <code>arr</code> the task is to find: </p><ul><li>two <code>nonnegative</code> integers A and B such as <code>P = A ** 2 + B ** 2 (1)</code>.</li></ul><p>The function <code>solve(arr)</code> should return an array or a list <code>[A, B]</code> or a tuple where A and B verify (1).</p><h4 id="examples">Examples:</h4><pre><code>solve([2, 1, 3, 4]) returns [2, 11] :(2*2 + 1*1) * (3*3 + 4*4) = 5 * 25 = 125 and 2 * 2 + 11 * 11 = 125solve([2, 1, 3, 4, 2, 2, 1, 5, 2, 3, 4, 5]) returns [2344, 2892] :(2*2 + 1*1) * (3*3 + 4*4) * (2*2 + 2*2) * (1*1 + 5*5) * (2*2 + 3*3) * (4*4 + 5*5) = 13858000and 2344 * 2344 + 2892 * 2892 = 13858000</code></pre><h4 id="notes">Notes</h4><ul><li>The decomposition into <code>A ** 2 + B ** 2</code> is not unique: the testing function checks if (1) is verified.</li><li>Lengths of lists are less than 100 and elements of lists less than 30<pre><code>solve([21, 24, 15, 22, 1, 2]) can return [639, 1788] or [1431, 1248]; both return are correct.P is     3605265A*A+B*B  3605265</code></pre></li><li>Please ask before translating</li></ul>
