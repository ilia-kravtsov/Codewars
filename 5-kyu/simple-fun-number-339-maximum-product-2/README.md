<h2><a href=https://www.codewars.com/kata/595bd047b96bed1f59000001/train/javascript target="_blank">Simple Fun #339: Maximum Product 2</a></h2><h3>5 kyu</h3><h1 id="task">Task</h1><p>Given an positive integer <code>n(5 &lt;= n &lt;= 100)</code>. Your task is to divide <code>n</code> into some <code>different</code> integers, maximize the product of these integers. </p><h1 id="example">Example</h1><p>For <code>n = 5</code>, the output should be <code>6</code>.</p><p><code>5 can divide into 2 and 3, 2 x 3 = 6</code></p><p>For <code>n = 8</code>, the output should be <code>15</code>.</p><p><code>8 can divide into 3 and 5, 3 x 5 = 15</code></p><p>For <code>n = 10</code>, the output should be <code>30</code>.</p><p><code>10 can divide into 2, 3 and 5, 2 x 3 x 5 = 30</code></p><p>For <code>n = 15</code>, the output should be <code>144</code>.</p><p><code>15 can divide into 2, 3, 4 and 6, 2 x 3 x 4 x 6 = 144</code></p>
